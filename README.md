# Galaxy Invader

A retro-style arcade space shooter built with HTML5 Canvas and Vanilla JavaScript.

## How to Play
1. **Open the Game**: Double-click `index.html` in your browser.
2. **Controls**:
   - **Arrow Keys**: Move Ship
   - **Spacebar**: Shoot
3. **Goal**: Destroy enemies, avoid collision, and survive as long as possible!

## How to Run Locally
### Option 1: Direct Open
Simply double-click the `index.html` file. It should open in your default web browser.

### Option 2: VS Code Live Server (Recommended)
If you are using VS Code:
1. Install the "Live Server" extension.
2. Right-click `index.html` and select "Open with Live Server".
This will automatically reload the game when you make changes.

## How to Share
### Option 1: Zip File
Compress the entire `galaxy_invader` folder into a `.zip` file and send it to your friends. They just need to unzip it and open `index.html`.

### Option 2: GitHub Pages (Free Web Hosting)
1. Create a repository on GitHub.
2. Upload these files.
3. Go to Settings > Pages and enable GitHub Pages.
4. Send the generated link to anyone!

### Option 3: Netlify Drop
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the `galaxy_invader` folder.
3. It will give you a live URL instantly.
